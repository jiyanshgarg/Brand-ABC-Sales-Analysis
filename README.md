## Project Overview
This comprehensive Tableau dashboard provides an in-depth analysis of Brand ABC's sales performance across multiple dimensions. The interactive visualization offers stakeholders a powerful tool to monitor key business metrics, identify trends, and make data-driven decisions to optimize sales strategies and drive revenue growth.

## Dashboard Features
### Key Performance Indicators (KPIs)
* Total Sales: $1.07M overall revenue performance
* Total Quantity: 3.84K units sold across all regions
* Average Sales: $278.73 per transaction
* Total Products: 24 unique products in portfolio
